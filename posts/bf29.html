段落标签和换行标签

效果

“观星”采用LTE-V通信模组阵列，可实时抓取V2X空口数据，并对消息数据进行解析分析，完成规范性/数据一致性检测，并做可视化处理，对消息接收强度（dBm）、收/丢包率、通信距离等性能参数进行实时监测，对周边环境中各类设备与各类V2X消息进行统计。

同时，“观星”还支持对测试过程进行录制与回放，保证测试的可复现性，便于问题追溯；支持录制过程中的事件断点插入，便于回放时快速定位关键事件点。

此外，“观星”支持从空间/时间维度对测试场、道路周边V2X服务覆盖程度进行分析（综合消息数量、消息种类、接收强度、丢包率），绘制服务覆盖热力图。便于运营单位的日常设备运维，指导路测设备铺设规划。

“观星”的推出可切实解决测试园区、开放道路的设备运维与实车测试等业务需求。

图像标签和路径

图像标签的属性

主要的一部分

属性 属性值 必需 说明
src URL 是 可能的值：绝对 URL - 指向其他站点（比如 src=”http://www.example.com/"） 相对 URL - 指向站点内的文件（比如 src=”/i/image.gif”）
alt text 是 替换文本，图像不能显示时才显示的文字
title text 否 提示文本，鼠标放到图像上时显示的文字
width pixels % 否 设置图片的宽度,如400 或 100%
height pixels % 否 设置图片的高度,如400 或 100%
border pixels 否 不推荐使用，设置图片的边框粗细，如10

路径

相对路径

以引用文件所在目录为参考，而建立的目录。（相对于HTML文件的位置）

相对路径分类 符号 说明
同一级路径 和HTNL文件处于同一目录，如<img src="jpg.jpg" />
下一级路径 / 处于HTNL文件下一级目录，如<img src="img/jpg.jpg" />
上一级路径 ../ 处于HTNL文件上一级目录，如<img src="../jpg.jpg" />

绝对路径

例如：src=”http://www.example.com/" src=”D:/img/image.gif”

超链接标签

链接的语法格式

文本或图像等

属性 属性值 必需 说明
href URL , 锚链接 是 指定链接目标的URL地址或锚点,可以是外部链接或内部链接或锚点（href=”#id”）。
target _self , _blank 否 指定链接页面的打开方式，_self当前窗口打开，_blank新窗口打开。
